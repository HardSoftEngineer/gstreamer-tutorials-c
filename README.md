# gstreamer-tutorials-c

This repository contains my walkthrough and implementation of the official C tutorials from the GStreamer documentation.

The goal is to go through the examples step by step, understand the core concepts of the framework, and build a practical foundation.

Official tutorials:

https://gstreamer.freedesktop.org/documentation/tutorials/basic/index.html?gi-language=c
